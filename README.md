## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- No installation or dependencies required!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/tagadearpit/my-calculator.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd my-calculator
   ```

3. **Open in your browser**
   - Double-click `index.html`, or
   - Right-click and select "Open with" → Your preferred browser

That's it! 🎉

## 💻 Usage

### Basic Operations

1. **Click number buttons** to input numbers
2. **Click operator buttons** (+, −, ×, ÷) to select operations
3. **Click equals (=)** to see the result
4. **Click AC** to clear all entries

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Input numbers |
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `Enter` or `=` | Calculate result |
| `C` | Clear all |
| `Backspace` | Delete last digit |

## 🎯 How It Works

1. **Input Phase** - User enters numbers and selects operations
2. **Display Phase** - Real-time display updates as you type
3. **Calculation Phase** - When equals is pressed, JavaScript processes the calculation
4. **Result Phase** - Answer is displayed and ready for next calculation
